# Changelog

All notable changes to the "specdd" extension will be documented in this file.

## [1.0.0] - 2026-06-04

### Added

- Add SpecDD language support for `.sdd` files.
- Add parser-backed diagnostics, completions, symbols, folding, hovers, exact path links, glob links, and related-spec
  commands.
- Add richer SpecDD highlighting for section categories, task states, scenario steps, key-value entries, paths, globs,
  symbol references, and inline code spans.
- Add glob reference resolution with support for `*`, `?`, `[abc]`, `{a,b}`, `**`, and `**/`.
- Add SpecDD extension and `.sdd` file icons.
- Add homepage, repository, and issue tracker metadata.
- Add TypeScript, Jest, VSIX packaging, and GitHub release workflow.
